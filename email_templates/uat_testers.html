Dear Tester

Thank you for participating in user acceptance testing for Budget plan app, download and install app onto your device and test the listed features and submit a bug report if you come across any bugs.

Features

  • Registration flow (setup account -> select currency, input income, allow debt, select budget plan day)
  • add,update and delete income
  • display total of all incomes
  • add, update and delete expense groups and expenses
  • display total of all expenses
  • add, update and delete payment reminders
  • display of Payment reminders in home screen
  • display of Expense group shares in home screen
  • Process death safe (this means if you open the app, and minimize the app and come back to it later, it won't restart, it will continue where it left off.)
  • Tablet support for all screens.

Known bugs/issues still to be fixed

  • Expense Categories shares behaves weird when we have 1 expense category with no expenses.
  • Budget screen loads up slow on some devices.

Version: 0.0.3

Note: only the listed features are working, the rest (if showing any data) is just dummy data. Feel free to ask any questions when stuck.
